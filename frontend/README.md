# Smart Expense Analytics Dashboard

A **full-stack expense tracking and analytics dashboard** that helps users track spending, analyze financial patterns, and manage budgets efficiently.

The application allows users to **add expenses, categorize spending, visualize data through charts, and monitor their budget usage with smart analytics**.

---

# 🚀 Features

* Add, edit, and delete expenses
* Category-wise spending analysis
* Monthly spending trends visualization
* Budget monitoring with dynamic progress bar
* Top spending categories detection
* Month-to-month spending comparison
* Average daily spending insights
* Smart spending recommendations
* Interactive charts and analytics dashboard

---

# 🧠 Analytics Features

The dashboard provides useful financial insights such as:

* Weekend vs weekday spending comparison
* Highest spending category detection
* Top 3 spending categories
* Average daily spending calculation
* Month-to-month spending change
* Budget usage progress indicator
* Spending recommendations based on budget usage

# ⚙️ Installation & Setup

## Clone the repository

```
git clone https://github.com/yourusername/smart-expense-analytics-dashboard.git
cd smart-expense-analytics-dashboard
```

---

## Backend Setup (FastAPI)

```
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

Backend runs on:

```
http://127.0.0.1:8000
```

---

## Frontend Setup (React)

```
cd frontend
npm install
npm run dev
```

Frontend runs on:

```
http://localhost:5173
```
